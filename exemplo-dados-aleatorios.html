Exemplo - Geração de Dados Aleatórios

Demonstração: Este exemplo mostra como o sistema gera automaticamente dados aleatórios do cliente para o PIX.
Dados do Cliente

Nome:

Email:

Telefone:

CPF:

Informações
  • CPF válido com dígitos verificadores
  • Telefone com DDD brasileiro
  • Email com domínios reais
  • Nome brasileiro realista
Como Funciona:
  1. Nome: Seleciona aleatoriamente de uma lista de nomes brasileiros
  2. Email: Gera baseado no nome + número aleatório + domínio
  3. Telefone: DDD brasileiro + 9 dígitos aleatórios
  4. CPF: 9 dígitos aleatórios + cálculo dos dígitos verificadores
